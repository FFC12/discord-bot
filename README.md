### Features
- [x] Moderation
- [x] Fun
- [x] Utility
- [x] Music
- [x] Video

### Installation
1. Clone the repo
   ```sh
   git clone

2. Create a `.env` file and fill it with the following
   ```env
   DISCORD_TOKEN=your_token
   BARD_TOKEN=your_prefix
   ```
3. Deploy by using [Heroku](https://heroku.com) or [Okteto](https://okteto.com) (docker-compose)
